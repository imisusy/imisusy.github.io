I won’t stay afraid，

我不会像这样一直害怕，

I had this vision that has got me worried，

我仿佛已经看到这令我担心的画面了，

‘Cause everyone wants someone，

因为每个人都想有个能属于自己的另外一半，

That’s one cliché that’s true，

这是陈词滥调但也确实说的对，

Except you I want no one，

于我而言除了你我谁也不想要，

Unless that someone’s you，

除非是个像你一样的某人吧，

It looks like you，

她看上去要像你，

Feels like you，

感觉起来要像你，

Smiles like you，

笑起来也要像你，

I want someone just like you，

但说到底我还是想要一个和你一模一样的人，

Forever blue，

但我估计会一直忧伤下去，

‘Cause there’s no one else like，

因为这里根本没有一个像你这样的人，

Been spending my whole life waiting，

于是我将继续用我一生来等待，

For someone just like you，

等待一个像你一样的某人，

I want you in my heart，

我想要你住在我心里，

I see you in my dreams，

出现在我的梦里，

I know I make you mad，

我知道我可能要把你逼疯了，

As crazy as it seems，

但我对你确实就是这么疯狂，

There’s no one else，

恐怕再无他人，

There’s no one else，

真的再无他人，

There’s no one else，

一个都没有，

There’s no one else，

真的一个都没有，

I need your help，

我需要你来解救我，

‘Cause you’re so far，

但是你现在如此遥远，

You’re so far，

感觉冷淡，

‘Cause you’re so far，

你现在如此遥远，

You’re so far，

态度冷漠，

‘Cause you’re so far，

是因为距离遥远吗，

‘Cause you’re so far，

还是你想保持距离，

‘Cause you’re so far，

你真的离我太远了，

You’re so far，

冷漠的你呀，

You’re so far，

够不到的你
